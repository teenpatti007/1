Watch Free Movies on Kodi

MovieHub is a Kodi video addon with a curated movie & web-series catalogue. Access is protected by a 4-digit subscription passcode that is locked to a single device.

Get Started Admin Panel

🔍 Browse & Search

Search, Latest, Genres and Categories — all from your couch.

▶ One-Click Play

Resolves streams directly inside Kodi with inputstream.adaptive (HLS).

🔐 Subscription Codes

4-digit passcodes issued from the admin panel, each locked to one device.

📺 Any Device

Windows, Linux, macOS, Android, LibreELEC, OSMC — anywhere Kodi runs.

Install on Kodi

1
Download the addon zip from the releases page (or the link your provider gave you).
2
In Kodi go to Settings → Add-ons → Install from zip file and select the downloaded plugin.video.moviehub.zip.
3
Open MovieHub from your Add-ons. When prompted, enter the 4-digit access code you were given. The code is linked to this device only.
4
(Optional) Enable auto-updates by installing the MovieHub Repository first, then MovieHub from it.

Tip: If you ever reinstall Kodi on the same device your code keeps working. Using the same code on a different device will be rejected.

MovieHub is an independent project. Streaming copyrighted content may be illegal in your country — use at your own risk.
